# Unyt Marketplace -- Mycelium listings

This is a prototype for interacing with the cloud hosting listings from the Mycelium Network to render them inside the Unyt marketplace. The UI includes pagination, and filters, but needs enhancements for ease of use.

## Instructions to run/test locally

In a terminal, run:

```bash
npm install
```
Followed by:
```bash
npm run dev
```
You should see a link where the page is being rendered. Either Ctrl-click that link or copy it into your browser.
